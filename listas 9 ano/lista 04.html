Questões sobre o discriminante (Δ)

1) Observe a equação:

x² - 4x + 4 = 0

Calcule o discriminante (Δ).

Depois responda:

  1. Qual é o valor de Δ?
  2. Quantas raízes reais essa equação possui?

2) Determine a natureza das raízes sem resolver a equação

Use apenas o discriminante para dizer se a equação possui:

  • Duas raízes reais diferentes;
  • Uma raiz real dupla;
  • Ou nenhuma raiz real.
  1. x² - 6x + 8 = 0
  2. x² - 2x + 1 = 0
  3. x² + 2x + 5 = 0

3) Resolva as equações do 2º grau

  1. x² - 5x + 6 = 0
  2. x² + 7x + 12 = 0
  3. 2x² - 8x + 6 = 0
  4. 3x² + x - 2 = 0
  5. x² - 9 = 0
