Atividade – Equação do 2º Grau

Parte 1 – Identifique os coeficientes a, b e c

Forma geral: ax² + bx + c = 0

  1. x² + 5x + 6 = 0
  2. 2x² − 3x + 1 = 0
  3. 3x² + 4 = 0
  4. −x² + 7x = 0
  5. 4x² − 9 = 0
  6. x² − x − 12 = 0
  7. 5x² + 2x = 0
  8. −2x² + 3x − 8 = 0
  9. x² + 9 = 0
  10. 6x² − 11x + 3 = 0

Parte 2 – Complete ou Incompleta

  1. x² + 5x + 6 = 0
  2. 3x² + 4 = 0
  3. 2x² − 3x = 0
  4. x² − 9 = 0
  5. 4x² + x + 1 = 0
  6. −x² + 7x = 0
  7. 5x² − 10 = 0
  8. x² − x − 12 = 0
  9. 6x² + 2x = 0
  10. −2x² + 8 = 0

Observação: Completa: possui a, b e c diferentes de zero. Incompleta: falta b ou c (ou ambos).
